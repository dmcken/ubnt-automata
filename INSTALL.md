# Installation


## Pip

### Local development

Windows:
```
git clone https://github.com/dmcken/ubnt_automata.git
cd <git root dir>
python -m pip install -e .
```

Unix:
```
git clone https://github.com/dmcken/ubnt_automata.git
cd <git root dir>
python3 -m pip install --upgrade .
```

### Git

```
pip install git+https://github.com/dmcken/ubnt_automata.git
```
